# Salma & Khadija — Summer Bridge to Grade 7

An ESL-friendly interactive summer prep page for two rising 7th graders, built on the [LearnKit universal learning engine](https://zidni0.github.io/interactive-learning-template/).

**Live site:** https://zidni0.github.io/salma-khadija-summer/

## Subjects

- **Math** — integers, ratios & proportions, expressions & equations, geometry, statistics & probability
- **English / ELA** — reading comprehension, paragraph writing, argumentative essays, informative essays, language & vocabulary

## How to use

1. Open the live site.
2. Pick a subject for the day: **Math** or **English**.
3. Work through the lessons at your own pace.
4. Each lesson has a lesson block, drills, a practice game, and an activity.
5. Click **Switch Subject** when you are ready to switch.
6. Lessons are not locked — you can jump to any lesson.

## Built for ESL learners

- All text can be read aloud using the 🔊 buttons.
- Simple vocabulary with definitions.
- Lots of examples before exercises.
- Sentence frames for writing.
- Visual inline SVG illustrations.

## Tech stack

- Vanilla JavaScript + CSS
- No build step
- Deployed to GitHub Pages

Edit `content.js` to change lessons, add more units, or adjust the theme. The engine is shared from `interactive-learning-template` and should not be edited directly here.
